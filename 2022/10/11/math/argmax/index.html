argmax

Claire Lin
11-10-2022
數學

The argmax function returns the argument or arguments (arg) for the target function that returns the maximum (max) value from the target function.

g(x) = x * 2

g(1) = 1 * 2 = 2
g(2) = 2 * 2 = 4
g(3) = 3 * 2 = 6
g(4) = 4 * 2 = 8
g(5) = 5 * 2 = 10

$\underset{x}{\operatorname{\argmax}} g(x) = 5$
argmax(g(x)) = 5

argument 是 g(x) 中的 x ，argmax 要得到的是能使 g(x) 變成最大值的 x
