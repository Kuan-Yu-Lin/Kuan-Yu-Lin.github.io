部落格創建_始

Posted on 21-10-2022 In 崩潰記事

從早上九點開始，做了八個多小時才成功，系統更新是好事，但可能需要不一樣的操作步驟，所以常常差一步就可以成功。

最一開始 node.js 的版本問題就卡超級久，怎麼更新都沒用，版本永遠卡在 v8.17.0 ，到底是什麼設定這麼瞎，也查不到解決方法。

最後，往其他方向查，終於找到一個可行的解決方法，懂得查資料真的很重要。

前往目標資料夾(blog)：
nvm install 16.18.0(官網上最新版本)
（此時版本會在這個資料夾改成v16最新版，這樣才能用npm）

在_config.yml中，適合我的Hexo佈署(deploy)路徑，用ssh：
deploy:
type: git
repo: <repository url> #git@github.com:…/….github.io.git
branch: gh-pages

這個週末沒有活動，原本一週課間的讀書時間也都先刪去，稍微適應並決定要怎麼安排課程，晚點安排出哪幾天需要提早準備的午餐和早餐，未確定的週間功課也都確定下來，排出固定行程。
